--- title: Bootstrapping Design description: Bootstrapping Design created: !!timestamp '2013-09-01 10:00:00' tags: - development --- {% mark excerpt -%} Efficiently is a site for creating [shareable online outlines](https://eff.iciently.com/login). I use it every day because breaking my work into [tasks](https://eff.iciently.com/login) and [subtasks](https://eff.iciently.com/login) makes me much more productive. You can try the [Efficiently demo](https://eff.iciently.com/docview/r/4ed70c031a7ba32edb000002/). I've never been good at design - I'm still not, but I wanted to see just how non-terrible I could be. When I first built Efficiently, it was called Yata - Yet Another Task Application. I like acronyms, and yata.in was a domain that was available for cheap. I built it, set it up, and it was fine... but the design left a little to be desired. Here is the landing page: ![]({{ media_url('images/yatasplash.png') }}) And the actual application ![]({{ media_url('images/yataoutline.png') }}) So it's certainly not as terrible as it could be, but it wasn't great. I saw a post on hackernews about [bootstrapping design](http://bootstrappingdesign.com/), which was a way to learn some basics about design for hackers. It's a great book, a quick read, and is responsible for the way the site looks today. It was pretty simple, so I wanted to share how anyone can use the simple design tips from [bootstrapping design](http://bootstrappingdesign.com/) to improve their site.

Typography

Typography is emphasized in the book, and it's super important. The first thing I did was browse google web fonts and select a few that I liked. [bootstrapping design](http://bootstrappingdesign.com/) stressed that you want to use a few fonts, but you don't want to go crazy. I settled on 3 fonts, Lobster Two for the logo, Open Sans for regular text, and Open Sans Condensed for headings. See for yourself

Choosing a Color Theme

{%- endmark %} I went to a website that had a bunch of color themes, I don't remember which one, but it was pretty easy to select one that I liked. After selecting it, I simply used all the color codes in my css. Efficiently is a very text heavy site and it's not super colorful, however having a cohesive and uniform approach to all accent colors is a good idea

Creating a Logo

One of the best things pointed out by bootstrapping design, is that you can get great logos by choosing great fonts! I went with Lobster Two, and then created some a simple bullet list using SVG and I was done really quickly. I wrote the SVG by hand, but I'm sure there are alot of tools out there for doing so.
