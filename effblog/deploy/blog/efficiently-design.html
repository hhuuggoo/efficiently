Bootstrapping Design

Tweet

Sun, 01 Sep 2013

Efficiently is a site for creating shareable online outlines. I use it every day because breaking my work into tasks and subtasks makes me much more productive. You can try the Efficiently demo. I’ve never been good at design - I’m still not, but I wanted to see just how non-terrible I could be. When I first built Efficiently, it was called Yata - Yet Another Task Application. I like acronyms, and yata.in was a domain that was available for cheap. I built it, set it up, and it was fine… but the design left a little to be desired. Here is the landing page:

And the actual application

So it’s certainly not as terrible as it could be, but it wasn’t great. I saw a post on hackernews about bootstrapping design, which was a way to learn some basics about design for hackers. It’s a great book, a quick read, and is responsible for the way the site looks today. It was pretty simple, so I wanted to share how anyone can use the simple design tips from bootstrapping design to improve their site.

Typography

Typography is emphasized in the book, and it’s super important. The first thing I did was browse google web fonts and select a few that I liked. bootstrapping design stressed that you want to use a few fonts, but you don’t want to go crazy. I settled on 3 fonts, Lobster Two for the logo, Open Sans for regular text, and Open Sans Condensed for headings. See for yourself

Choosing a Color Theme

I went to a website that had a bunch of color themes, I don’t remember which one, but it was pretty easy to select one that I liked. After selecting it, I simply used all the color codes in my css. Efficiently is a very text heavy site and it’s not super colorful, however having a cohesive and uniform approach to all accent colors is a good idea

Creating a Logo

One of the best things pointed out by bootstrapping design, is that you can get great logos by choosing great fonts! I went with Lobster Two, and then created some a simple bullet list using SVG and I was done really quickly. I wrote the SVG by hand, but I’m sure there are alot of tools out there for doing so. There you have it, a few simple design tips from bootstrapping design, and my site is looking alot better. It’s worth noting that using less and bootstrap made it alot easier to put this site together, however I really think updating your typography, color scheme, and logo can make a website much better.
